Free-Space Management

The simulator uses a bitmap-style structure where each block is marked as free, used, deleted, recovered, or corrupted. This shows how an OS tracks available storage.

Directory Structure

Files are arranged under folders such as root, documents, images, and Recovered. During recovery, deleted files are restored into the Recovered folder.

File Access Methods

Contiguous allocation stores files in continuous blocks, linked allocation connects scattered blocks, and indexed allocation stores block addresses using an index idea.
